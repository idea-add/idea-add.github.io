## 한눈에 보는 정의

**SQL**(Structured Query Language)은 관계형 데이터베이스에 저장된 데이터를 조회하고, 추가·수정·삭제하기 위해 만들어진 표준 질의 언어입니다.

## 자세히 알아보기

- 예문: "`SELECT * FROM users WHERE age > 20;`처럼 **SQL** 문 하나로 조건에 맞는 데이터를 조회할 수 있다."
- 테이블 구조 자체를 만들고 바꾸는 데 쓰는 {{< term "ddl" >}}(데이터 정의어)과, 실제 데이터를 조회·조작하는 데 쓰는 {{< term "dml" >}}(데이터 조작어)로 나뉩니다.
- 특정 회사의 제품이 아니라 국제 표준으로 정해져 있어, 데이터베이스 제품이 달라도 기본적인 SQL 문법은 대부분 비슷하게 통합니다.
- 웹 서비스 백엔드부터 데이터 분석까지, 데이터를 다루는 거의 모든 개발·분석 업무에서 기본적으로 요구되는 핵심 기술입니다.

## 관련 용어

- {{< term "dml" >}}
