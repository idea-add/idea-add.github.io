## 한눈에 보는 정의

**반복문**(Loop Statement)은 정해진 조건을 만족하는 동안 같은 코드 블록을 여러 번 반복해서 실행하는 프로그래밍 문법으로, 대표적으로 `for`문과 `while`문이 있습니다.

## 자세히 알아보기

- 예문: "`for (i = 0; i < 10; i++) { ... }`처럼 **반복문**을 쓰면 같은 코드를 10번 반복해서 실행할 수 있다."
- `for`문은 반복 횟수가 정해져 있을 때, `while`문은 특정 조건이 참인 동안 계속 반복해야 할 때 주로 사용합니다.
- {{< term "conditional-statement" >}}(조건문)과 결합해, 반복 중 특정 조건에서만 반복을 멈추거나 건너뛰도록 만들 수 있습니다.
- 리스트나 배열에 담긴 데이터를 하나씩 꺼내 처리할 때도 반복문이 필수적으로 쓰이며, 이는 대부분의 프로그램에서 가장 자주 등장하는 문법 중 하나입니다.
