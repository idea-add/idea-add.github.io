## 한눈에 보는 정의

**DDL**(Data Definition Language, 데이터 정의어)은 {{< term "sql" >}}에서 데이터베이스의 테이블 같은 구조 자체를 만들고(CREATE), 변경하고(ALTER), 삭제하는(DROP) 데 쓰이는 명령어 집합입니다.

## 자세히 알아보기

- 예문: "`CREATE TABLE users (...)`는 users라는 새 테이블 구조를 만드는 **DDL** 명령이다."
- 실제 데이터 값이 아니라, 테이블의 이름과 열(칼럼) 구성 같은 데이터의 '틀' 자체를 정의한다는 점이 핵심입니다.
- 테이블 안의 실제 데이터를 조회·추가·수정·삭제하는 {{< term "dml" >}}(데이터 조작어)과는 역할이 뚜렷하게 구분됩니다.
- DDL로 테이블 구조를 바꾸면 그 안에 저장된 기존 데이터에도 영향을 줄 수 있어, 운영 중인 시스템에서는 신중하게 다뤄야 합니다.
