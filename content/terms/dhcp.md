## 한눈에 보는 정의

**DHCP**(Dynamic Host Configuration Protocol, 동적 호스트 구성 프로토콜)는 네트워크에 새로 연결된 장비에게, 사람이 일일이 설정하지 않아도 {{< term "ip-address" >}}(IP 주소)를 자동으로 할당해주는 프로토콜입니다.

## 자세히 알아보기

- 예문: "카페의 Wi-Fi에 접속하면 **DHCP** 덕분에 별도 설정 없이도 곧바로 인터넷을 쓸 수 있다."
- {{< term "router" >}}(공유기 등 라우터)가 DHCP 서버 역할을 맡아, 새로 연결되는 장비마다 사용 가능한 IP 주소를 자동으로 나눠주고 회수합니다.
- IP 주소를 관리자가 일일이 수동으로 설정해야 했던 과거 방식에 비해, 장비가 늘어나거나 자주 바뀌는 환경에서 관리 부담을 크게 줄여줍니다.
- 보통 일정 기간(임대 기간) 동안만 IP 주소를 할당하고, 기간이 지나면 회수해 다른 장비에 다시 배정할 수 있도록 관리합니다.
