## 한눈에 보는 정의

**DNS**(Domain Name System, 도메인 네임 시스템)는 example.com처럼 사람이 기억하기 쉬운 도메인 이름을, 컴퓨터가 실제로 접속하는 데 필요한 {{< term "ip-address" >}}(IP 주소)로 변환해주는 시스템입니다.

## 자세히 알아보기

- 예문: "브라우저에 주소를 입력하면, **DNS**가 그 도메인에 해당하는 서버의 IP 주소를 찾아 실제 접속을 연결해준다."
- 숫자로 된 IP 주소를 일일이 외우는 대신, 사람이 기억하기 쉬운 이름으로 웹사이트에 접속할 수 있게 해주는 인터넷의 '전화번호부' 같은 역할을 합니다.
- 도메인 이름과 IP 주소의 짝을 저장해두는 여러 단계의 서버를 거쳐 최종 IP 주소를 찾아내는 방식으로 동작하며, 한 번 찾은 결과는 일정 시간 저장(캐싱)해 다음 조회 속도를 높입니다.
- DNS 서버가 장애를 일으키면 도메인 이름으로 접속이 안 되는 문제가 생길 수 있어, 여러 대의 DNS 서버를 함께 운영해 안정성을 높이는 경우가 많습니다.
