## 한눈에 보는 정의

**EFS**(Elastic File System)는 여러 대의 {{< term "ec2" >}}(EC2) 서버가 동시에 마운트해 같은 파일을 함께 읽고 쓸 수 있는 클라우드 공유 파일 저장 서비스입니다.

## 자세히 알아보기

- 예문: "여러 대의 서버가 같은 사용자 업로드 파일에 접근해야 하는 서비스는, 서버마다 따로 저장하는 대신 **EFS**를 공유 저장소로 활용한다."
- 원칙적으로 하나의 서버에만 전용으로 붙는 {{< term "ebs" >}}(EBS)와 달리, EFS는 여러 서버가 마치 같은 폴더를 보는 것처럼 동시에 접근할 수 있도록 설계되었습니다.
- 저장하는 데이터 양에 따라 용량이 자동으로 늘어나거나 줄어들어, 미리 용량을 정해두고 관리할 필요가 없습니다.
- 여러 서버로 트래픽을 분산하는 {{< term "elb" >}}(ELB) 환경에서, 어느 서버가 요청을 처리하든 같은 파일에 접근해야 할 때 특히 유용하게 쓰입니다.
