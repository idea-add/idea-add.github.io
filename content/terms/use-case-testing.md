## 한눈에 보는 정의

**유스케이스 테스팅**(Use Case Testing)은 실제 사용자가 시스템을 사용하는 일련의 시나리오(유스케이스)를 처음부터 끝까지 그대로 따라가며 검증하는 {{< term "black-box-testing" >}}(블랙박스 테스트) 기법입니다.

## 자세히 알아보기

- 예문: "'회원가입 → 로그인 → 상품 검색 → 주문 → 결제'라는 전체 흐름을 실제로 따라가며 테스트하는 것이 **유스케이스 테스팅**이다."
- 개별 기능 하나하나를 따로 확인하는 대신, 여러 기능이 실제 업무 흐름 속에서 자연스럽게 이어지는지를 사용자 관점에서 검증하는 데 초점을 둡니다.
- 정상적으로 흘러가는 기본 시나리오뿐 아니라, 중간에 오류가 발생하거나 사용자가 예외적으로 행동하는 대안 시나리오까지 함께 테스트하는 것이 중요합니다.
- {{< term "acceptance-test" >}}(인수 테스트) 단계에서, 실제 업무 흐름이 요구사항대로 동작하는지 확인하는 데 자주 활용되는 기법입니다.
