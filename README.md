# ✈️ 가족여행 플래너 (Family Travel Planner) - WP 정적 생성 패키지

본 프로젝트는 구글 애드센스 승인 심사에 부합하는 고품질 정보성 브랜드 미디어 아키텍처와 워드프레스 레이아웃 감성을 정적 클라이언트 사이드 기술(HTML/CSS/JS)로 완벽히 마감한 완성형 패키지입니다.

## 🚀 Replit 즉시 실행 및 운용 방법
1. Replit 프로젝트 내부 루트 디렉토리에 명시된 구조대로 파일을 생성 및 복사합니다.
2. Replit 상단의 **[Run]** 버튼을 클릭하면 내장 라이브 서버가 기동되며 프론트엔드가 활성화됩니다.
3. 임의의 문서 하단이나 푸터에서 책임 편집자 이름인 **[김대형]**을 클릭하면 `/author/` 칼럼 허브로 연결됩니다.
4. 상단 네비게이션의 **[⚙ 관리자]** 링크를 클릭하여 가상 로그인하면 `localStorage` 기반의 데이터 제어판(CMS-lite)이 활성화되며 즉시 데이터 수정 및 신규 칼럼 추가 기능이 해제됩니다.

## ✍️ 핵심 환경 변수 및 콘텐츠 튜닝 위치 안내

*   **브랜드명 및 마스터 설정 변경:** 
    `assets/js/db.js` 파일의 상단 `SiteConfig` 오브젝트 구조체 내의 `name`, `tagline`, `topic` 등을 변경하면 홈 화면, 헤더, 푸터 전체 레이아웃에 실시간 연동되어 동적 반영됩니다.
*   **브랜드 메인/서브 컬러 테마 커스텀:** 
    `assets/css/style.css` 파일의 최상단 `:root` 가상 선택자 영역 내부의 `--main-color` 및 `--sub-color` 해시코드를 변경하면 사이트 전체에 설정된 톤앤매너 색조가 일괄 변경됩니다.
*   **공식 수신 이메일 주소 수정:** 
    `assets/js/db.js` 의 `SiteConfig.contactEmail`을 수정하시면 푸터 하단 및 문의하기 폼의 수신 유도부 데이터가 자동 개편됩니다.
*   **지식 글 15개 및 에디터 칼럼 3개 원문 교체:** 
    `assets/js/db.js` 내부의 배열 데이터 `PostsData` 및 `ColumnsData` 배열 원소들을 수정하거나 신규 오브젝트를 형식에 맞춰 인젝션해 주면 정적 웹사이트 엔진에 클린 결합됩니다.
