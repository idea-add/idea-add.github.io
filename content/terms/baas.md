## 한눈에 보는 정의

**BaaS**(Backend as a Service)는 로그인, 데이터베이스, 파일 저장, 푸시 알림처럼 앱에 흔히 필요한 백엔드 기능들을 미리 만들어 제공해, 개발자가 서버 쪽 코드를 직접 짜지 않고도 화면(프론트엔드) 개발에만 집중할 수 있게 해주는 서비스입니다.

## 자세히 알아보기

- 예문: "1인 개발자가 짧은 시간 안에 앱을 출시할 수 있었던 것은, 회원가입과 데이터 저장 기능을 직접 만들지 않고 **BaaS**를 가져다 썼기 때문이다."
- 회원 인증, {{< term "dbms" >}}(데이터베이스), 파일 업로드 같은 반복적으로 필요한 기능을 API 형태로 미리 제공해, 개발 속도를 크게 높여줍니다.
- {{< term "faas" >}}(FaaS)가 개별 함수 단위의 서버 기능을 제공하는 데 초점을 둔다면, BaaS는 앱 개발에 필요한 백엔드 기능 전반을 하나의 완성된 서비스 묶음으로 제공한다는 점에서 범위가 다릅니다.
- 빠른 프로토타입 제작이나 소규모 팀의 서비스 개발에서 특히 자주 채택됩니다.
