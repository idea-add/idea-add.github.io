## 한눈에 보는 정의

**CLI**(Command Line Interface, 명령줄 인터페이스)는 마우스나 아이콘 없이, 텍스트로 된 명령어를 직접 입력해 컴퓨터를 조작하는 방식입니다.

## 자세히 알아보기

- 예문: "개발자들은 파일을 옮기거나 서버에 접속할 때 **CLI**를 자주 활용한다."
- 클릭 몇 번으로 조작하는 {{< term "gui" >}}(그래픽 사용자 인터페이스)와 달리, 정확한 명령어와 옵션을 외우거나 찾아 입력해야 해서 배우는 데 시간이 걸리지만, 익숙해지면 반복 작업을 훨씬 빠르고 정확하게 처리할 수 있습니다.
- 같은 명령어를 여러 번 반복 실행하거나, 여러 명령어를 스크립트로 묶어 자동화하기에도 유리해 서버 관리나 {{< term "devops" >}} 업무에서 특히 많이 활용됩니다.
- 개발자용 도구나 서버 환경에서는 GUI가 아예 없이 CLI만 제공되는 경우도 흔합니다.
